## 🚀 使用步骤

### 第1步：启用新配置
将 `frequency_words_loan_ads.txt` 的内容复制到 `frequency_words.txt`：

**选项A（推荐）：完全替换**
```bash
# 备份原配置
cp config/frequency_words.txt config/frequency_words_backup.txt

# 启用新配置
cp config/frequency_words_loan_ads.txt config/frequency_words.txt
```

**选项B：保留测试**
- 先用 `frequency_words_loan_ads.txt` 测试几天
- 观察素材质量后决定是否正式替换

### 第2步：调整配置文件（可选）
编辑 `config/config.yaml`：

```yaml
report:
  mode: "incremental"  # 推荐：只推送新增新闻，避免重复
  max_news_per_keyword: 5  # 每个词组最多5条，控制信息量
  sort_by_position_first: true  # 按词组优先级排序

notification:
  enable_notification: true  # 启用推送
  push_window:
    enabled: true  # 启用推送时间窗口
    time_range:
      start: "09:00"  # 工作时间开始
      end: "21:00"    # 工作时间结束
    once_per_day: false  # 窗口内多次推送（实时获取素材）
```

### 第3步：测试运行
```bash
# 本地测试（如果已安装）
python main.py

# 或者触发 GitHub Actions
# 进入 Actions → Hot News Crawler → Run workflow
```

## 💡 使用建议

### 📝 脚本创作流程
```
1. 每天早上查看推送的新闻
   ↓
2. 筛选出3-5条适合的新闻
   ↓
3. 根据新闻类型选择对应的广告引子
   ↓
4. 套用脚本结构：新闻 + 引子 + 广告 + 结尾
```

### 📊 广告引子示例

| 新闻类型 | 新闻示例 | 广告引子 |
|---------|---------|---------|
| 民生经济 | "今年物价又涨了，老百姓压力大" | "生活成本确实高，有时候手头紧..." |
| 创业就业 | "越来越多人开始搞副业赚钱" | "创业初期最缺的就是启动资金..." |
| 消费金融 | "年轻人要学会理性消费和规划" | "但有时候确实需要应急周转..." |
| 节日消费 | "春节临近，置办年货开销大" | "过年人情往来确实是笔不小的支出..." |
| 租房生活 | "大城市租房押金动辄几万块" | "租房押金是一笔不小的负担..." |

### ⚙️ 优化技巧

**1. 定期调整关键词**
- 观察哪些词组产出素材较少 → 放宽限制
- 观察哪些词组产出素材不适用 → 增加过滤词

**2. 按需调整平台**
- 如果财经类新闻太专业 → 注释掉部分垂直财经平台
- 如果需要更多民生话题 → 启用网易新闻等综合平台

**3. 控制信息量**
```txt
# 在关键词组中添加 @数字 限制条数
物价
房租
油价
@3  # 这个词组最多显示3条
```

## 📈 预期效果

### 数据量预估
- **平台数量**：21个（原11个 + 新增10个）
- **关键词组**：7组（针对性筛选）
- **每日素材**：预计 **15-30 条**优质新闻（取决于热点）

### 素材质量
- ✅ **高相关性**：专注民生/财经/创业话题
- ✅ **强适配性**：易于嫁接贷款广告引子
- ✅ **零风险**：已过滤教育/医疗等受限话题

### 效率提升
- **以前**：每天浏览多个平台，手动筛选 → 耗时 1-2 小时
- **现在**：自动推送筛选后的新闻 → 10 分钟完成选题

## ⚠️ 注意事项

1. **政策合规**
   - 严禁使用教育类、医疗类话题
   - 避免过度渲染消费主义
   - 不使用恐慌性、负面性新闻

2. **内容审核**
   - AI 筛选后仍需人工二次审核
   - 确保新闻真实性
   - 避免敏感话题

3. **平台规则**
   - 遵守抖音广告投放规范
   - 注意金融类广告的合规要求
   - 定期更新关键词以适应政策变化
