# ========================================
# 📊 新闻适配度评分标准（人工判断参考）
# ========================================
#
# ⭐⭐⭐⭐⭐ 完美素材（优先使用）
# • 受众：90%以上的普通人都能感同身受
# • 场景：直接涉及缺钱、需要周转
# • 理解：一听就懂，无需解释
# 示例："多地房租上涨，年轻人直呼租不起"
#
# ⭐⭐⭐⭐ 优质素材（推荐使用）
# • 受众：70%以上的人群相关
# • 场景：间接涉及资金需求
# • 理解：稍加思考能理解
# 示例："副业成年轻人刚需，但启动资金是门槛"
#
# ⭐⭐⭐ 一般素材（谨慎使用）
# • 受众：50%左右的人群相关
# • 场景：需要一定引导才能关联借贷
# • 理解：需要一定背景知识
# 示例："央行调整利率政策"
#
# ⭐⭐ 勉强可用（不推荐）
# • 受众：30%以下的小众人群
# • 场景：与借贷关联弱
# • 理解：专业术语多
# 示例："金融监管新规出台"
#
# ⭐ 不适用（直接放弃）
# • 受众窄（特定行业/小众群体）
# • 场景无关（与资金需求无关）
# • 理解难（需要专业知识）
# 示例："某银行中介违规被查"
#
# ========================================
# 🎯 脚本创作黄金三问
# ========================================
#
# 在使用新闻前，问自己3个问题：
#
# 1️⃣ 受众够广吗？
#    ❓ 这个话题能覆盖多少人？
#    ✅ 房租上涨 → 所有租房的人
#    ❌ 银行中介被抓 → 只有金融从业者关心
#
# 2️⃣ 利益够直接吗？
#    ❓ 用户能立刻感受到和自己有关吗？
#    ✅ 工资延迟发放 → 立刻想到"没钱用了"
#    ❌ 货币政策调整 → 需要思考"这对我有啥影响"
#
# 3️⃣ 理解够简单吗？
#    ❓ 不需要解释就能听懂吗？
#    ✅ 物价上涨 → 人人都知道什么意思
#    ❌ LPR利率下调 → 需要解释什么是LPR
#
