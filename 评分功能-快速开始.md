## ⚙️ 进阶：启用自动评分

如果想让项目**自动过滤**低分新闻，编辑 `config/config.yaml`：

```yaml
report:
  # 其他配置...
  
  # 🎯 启用评分功能
  enable_scoring: true      # 改为 true
  min_score: 23             # 只保留23分以上的新闻
  show_score_in_report: true  # 在报告中显示评分
```

然后按照 `评分功能使用教程.md` 的"方式3"集成到main.py。

**注意**：这需要修改代码，新手建议先用 `batch_score.py` 手动评分。

## 💬 三个黄金问题（手动评分时用）

看到一条新闻，问自己：

1. ❓ **受众够广吗？**（能覆盖多少人？）
2. ❓ **利益够直接吗？**（立刻能感受到吗？）
3. ❓ **理解够简单吗？**（不需要解释就懂吗？）

三个都是"YES" → 立即使用 ✅  
有一个是"NO" → 谨慎使用 ⚠️  
两个以上"NO" → 直接放弃 ❌
